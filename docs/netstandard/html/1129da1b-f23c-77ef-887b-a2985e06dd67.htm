YamlLocalizationSourceBaseT Class

Represents a base class for a localization using embedded yaml files.
Copy
The yaml should have the following structure:

Attention-message
    en: Don't press the button.
    ja: ボタンを押さないでください。
Message2
    en: I am groot
    ja: 私はホートです
