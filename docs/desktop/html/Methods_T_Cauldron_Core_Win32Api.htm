Methods
  NameDescription
ActivateWindow
Brings the thread that created the specified window into the foreground and activates the window. Keyboard input is directed to the window, and various visual cues are changed for the user. The system assigns a slightly higher priority to the thread that created the foreground window than it does to other threads.
BroadcastMessage(UInt32)
The message is sent to all top-level windows in the system, including disabled or invisible unowned windows, overlapped windows, and pop-up windows; but the message is not sent to child windows.
BroadcastMessage(UInt32, IntPtr)
The message is sent to all top-level windows in the system, including disabled or invisible unowned windows, overlapped windows, and pop-up windows; but the message is not sent to child windows.
BroadcastMessage(UInt32, IntPtr, IntPtr)
The message is sent to all top-level windows in the system, including disabled or invisible unowned windows, overlapped windows, and pop-up windows; but the message is not sent to child windows.
ExtractAssociatedIcon
Extracts an icon from a exe, dll or ico file.
GetMessage
Retrieves the string sent by SendMessage(IntPtr, String).
GetMousePosition
Returns the mouse position on screen
GetUserTilePath
Gets the path of the user's profile picture
RegisterWindowMessage
Defines a new window message that is guaranteed to be unique throughout the system. The message value can be used when sending or posting messages.
SendMessage(IntPtr, String)
Sends the specified message string to a window. The SendMessage function calls the window procedure for the specified window and does not return until the window procedure has processed the message.
SendMessage(IntPtr, WindowsMessages, IntPtr, IntPtr)
Sends the specified message to a window or windows. The SendMessage function calls the window procedure for the specified window and does not return until the window procedure has processed the message.
