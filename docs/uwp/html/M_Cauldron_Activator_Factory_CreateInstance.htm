[This is preliminary documentation and is subject to change.]

Creates an instance of the specified type using the constructor that best matches the specified parameters. This method is similar to CreateInstance(Type, Object), but this takes the types defined with ComponentAttribute into account. This also executes the factory extensions (IFactoryExtension).
