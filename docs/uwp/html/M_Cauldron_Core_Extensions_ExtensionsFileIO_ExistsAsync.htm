When this method completes, it returns true if the file exists, otherwise false. If the caller does not have sufficient permissions to read the specified file, no exception is thrown and the method returns false regardless of the existence of path.

Usage Note

In Visual Basic and C#, you can call this method as an instance method on any object of type StorageFolder. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).
