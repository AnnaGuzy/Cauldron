JsonLocalizationSourceBaseT
Represents a base class for a localization using embedded json files.
Copy
The json should have the following structure:

[
    {
        "key" : "attention-message",
        "en" : "Don't press the button.",
        "ja" : "ボタンを押さないでください。"
    },
    {
        "key" : "message2",
        "en" : "I am groot",
        "ja" : "私はホートです"
    }
]
