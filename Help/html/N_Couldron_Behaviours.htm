ControlTemplateCommandFromControlNameBinding
Provides a behaviour that allows Buttons in ControlTemplate to bind to a Command that differs on every control.

Usecase example:

Two TextBoxex with ControlTemplates. Both TextBoxes shares the ControlTemplate. A Button resides in the ControlTemplate. The Button on each TextBox must invoke two different ICommands.
